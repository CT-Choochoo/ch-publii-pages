4. 变量名的规则

  1. 组成:必须是数字,字母,下划线组成
  2. 长度:任意长度,但是不建议太长,20字符以内
  3. 要求: 开头必须是字母
  4. 注意:区分大小写,有意义
# 变量名举例

#合法变量名
username = "1"
user_name = "1"
_user_name = "1"

#不推荐变量名
userName  = "1" # 不同于java ,python不推荐使用驼峰命名变量
abc0101 = "1" # 字段需要有有意义

#不合法变量名
01username = "1" # 开头必须是字母
user&name = "1" # 必须是数字,字母,下划线组成

5. 关键字

5.1 关键字的分类

python中关键字分为 强关键字和弱关键字

  • 强关键字 : 不允许变量或函数名定义, 解释器会抛出错误
  • 弱关键字 : 允许定义,但是会覆盖原有关键字,可能导致代码异常
